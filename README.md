# 🧠 Tasky Data Extractor

Automated scraper for extracting completed Tasky review data and uploading it to Google Sheets.

## 🚀 Features

- 🔗 Collects all Tasky task URLs with pagination
- 📄 Extracts:
  - Prompt
  - Response
  - User Sentiment
  - Issue Type
  - User Comment
- 📊 Uploads structured data directly to Google Sheets
- 🔁 Retry mechanism for Sheets API
- ⚡ Built with Playwright (async)

---

## 🛠️ Tech Stack

- Python
- Playwright (async browser automation)
- Google Sheets API (`gspread`)
- OAuth2 Service Account
